## Passos para rodar na sua máquina 🚀: 
1. Opção 1: Yarn 
  - Primeiro digite `yarn` e depois `yarn start`
2. Opção 2: Npm
  - Primeiro digite `npm` e depois `npm start`

## Tecnologias 🎯: 
* Chakra UI
* Nextjs
* API REST (Disney API)
* Axios
